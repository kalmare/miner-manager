# miner-manager
A manager of cpuminer for bitzeny build by Electron.
![SS](https://i.imgur.com/X9iLI0T.png)

## Requirement
- node
- npm
- electron

## Usage
```sh
$ npm run start
```

## Install
```sh
$ git clone https://github.com/kalmare/miner-manager/
$ cd miner-manager
$ npm install
```
